# Chatbot Kit Festa

Versão derivada do projeto de chatbot escolar, adaptada para empresas de aluguel de kit festa.

## Principais mudanças
- `Contatos` -> `Clientes`
- `Alunos` -> `Kits Festa`
- Novo catálogo de kits com tema, categoria, disponibilidade e valor
- Painel administrativo adaptado para locações
- Fluxo inicial de bot ajustado para catálogo, disponibilidade, valores, pagamento e atendimento humano

## Observação importante
Este projeto foi adaptado estruturalmente para o novo domínio. Como o banco antigo era escolar, você ainda deve gerar e aplicar uma nova migration Alembic antes de subir em produção.
